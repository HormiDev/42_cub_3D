# Trigonometría:

**radianes:** unidad de medida para angulos alternativa a los grados. 

**conversión:** 

-> 1 vuelta = 360 grados = 2π radianes

-> 3/4 vuelta = 270 grados = 3π/2 radianes

-> 1/2 vuelta = 180 grados = π radianes

-> 1/4 vuelta = 90 grados = π/2 radianes
